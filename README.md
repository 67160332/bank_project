# 🏦 Bank Insight AI - ระบบวิเคราะห์โอกาสการสมัครเงินฝาก ✨

แอปพลิเคชันสำหรับวิเคราะห์และพยากรณ์โอกาสที่ลูกค้าจะสมัคร "เงินฝากออมทรัพย์พิเศษ" โดยใช้เทคโนโลยี Machine Learning (AI) นำเสนอผ่าน Web Interface โทนสีชมพูพาสเทลที่สวยงามและใช้งานง่าย

##  จุดเด่นของโปรเจกต์
- **Real-time Prediction:** พยากรณ์ผลได้ทันทีเมื่อกรอกข้อมูลครบถ้วน พร้อมค่าความเชื่อมั่น (Confidence Score)
- **Cute Pastel UI:** ออกแบบด้วย Custom CSS โทนสีชมพู (#fff5f8) พร้อมเอฟเฟกต์ Balloons  และ Result Card 
- **Advanced Machine Learning:** ใช้ระบบ **Scikit-learn Pipeline** ในการจัดการข้อมูล (Preprocessing) เพื่อความแม่นยำสูงสุด

##  ข้อมูลที่ใช้ในการวิเคราะห์ (Customer Data )
เพื่อให้ระบบประมวลผลได้อย่างแม่นยำ ผู้ใช้งานต้องระบุข้อมูลลูกค้า 6 หัวข้อหลักดังนี้:
1. **อายุ (Age):** ช่วงวัยของลูกค้า 
2. **อาชีพ (Job Title):** กลุ่มอาชีพของลูกค้า (เช่น Management, Technician, Retired) 
3. **รายได้ต่อปี (Annual Income):** สถานะรายได้หลักเพื่อประเมินความสามารถทางการเงิน 
4. **เงินในบัญชี (Account Balance):** สภาพคล่องทางการเงินปัจจุบัน 
5. **สถานภาพ (Marital Status):** สถานะครอบครัว (Single, Married, Divorced) 
6. **การถือบัตรเครดิต (Credit Card):** ประวัติการใช้บริการผลิตภัณฑ์บัตรเครดิต 
*(ระบบมีการคำนวณร่วมกับค่า Credit Score และ History โดยอัตโนมัติในฐานข้อมูลส่วนหลังค่ะ)*

## 🛠️ เครื่องมือที่ใช้ (Tech Stack)
- **Language:** Python 3.11+
- **Machine Learning:** Pandas, Joblib, **Scikit-learn 1.6.1**
- **Frontend:** Streamlit (Custom CSS & Theme)
- **Deployment:** GitHub Repository & Streamlit Cloud

## 🚀 ขั้นตอนการใช้งาน
1. กรอกข้อมูลส่วนตัวและสถานะทางการเงินของลูกค้าในช่อง **"💌 ข้อมูลลูกค้า"**
2. ตรวจสอบความถูกต้องของข้อมูล (ข้อมูลครบถ้วนจะช่วยให้ AI แม่นยำขึ้น!)
3. กดปุ่ม **"🚀 เริ่มการวิเคราะห์เชิงลึก"**
4. ระบบจะแสดงผลการพยากรณ์ว่าลูกค้ามีแนวโน้มจะ **"สมัคร"** หรือ **"ไม่สมัคร"** พร้อมลูกโป่งฉลองความสำเร็จ
